Q.1 What is JavaScript. How to use it?

JavaScript is a high-level, interpreted programming language.
It is one of the three core technologies of the World Wide Web, along with HTML and CSS.
JavaScript is used to make web pages interactive and to add dynamic content.
It can also be used to create games, animations, and other multimedia applications.
Here are some examples of how JavaScript can be used:
To add interactivity to web pages, such as adding menus, forms, and games.
To validate forms before they are submitted.
To create animations and other visual effects.
To communicate with web servers.
To create mobile applications.

Q.3 Define a Data Types in js?

here are seven primitive data types in JavaScript:
String: A string is a sequence of characters.
Number: A number is a numeric value.
Boolean: A boolean is a value that can be either true or false.
Undefined: An undefined value is a value that has not been assigned yet.
Null: A null value is a value that represents the absence of a value.
Symbol: A symbol is a unique value that is used to identify an object.
BigInt: A BigInt is a number that is larger than the largest number that can be represented by a Number.
In addition to these primitive data types, there are also two complex data types in JavaScript:
Object: An object is a collection of properties.
Array: An array is an ordered list of values.
