Website:

A collection of many webpages that is hosted on web server.where a company,organization puts information that can be found on the World Wide Web

HTTP/HTTPS:

Hypertext transfer protocol is used to securely send data from a web browser to a website.

URL-URI/URN:

Uniform Resource Location-URL is a subset of URI that describes the network address or location where the source is available.A URL known as a web address, identifies a web resource and its location on a computer network. Uniform Resource Identifier-Sequence of characters used to identify resource location and name over the World Wide Web. Uniform Resource Name- formal naming that identifies a resource
